# Agent Process

Generated synthesis, non-authoritative.

1. Read `AGENTS.md`.
2. Pick Direct, Cold, or Broad/current-state profile.
3. Classify risk using `docs/DOMAIN_GATES.md` when the task touches production, data, security, compliance, billing, infrastructure, migrations, or shared seams.
4. Search with `rg`; read bounded ranges around relevant anchors.
5. Use GSD planning if the goal is broad or ambiguous.
6. Create a formal spec only when `docs/AGENT_OPERATING_MODEL.md` requires it.
7. Keep edits surgical and preserve dirty worktrees.
8. Verify by blast radius.
9. Stage explicit paths only when asked to commit.
10. Log friction if the rules or tools failed the task.
